# Control Ambiental Inteligente para Cabinas de Pestañas

## Descripción

Sistema IoT desarrollado con ESP32 para monitorear y controlar las condiciones ambientales dentro de una cabina de aplicación de extensiones de pestañas.

El sistema supervisa:

- Temperatura
- Humedad
- Calidad del aire

y activa automáticamente:

- Ventilador
- Humidificador

para mantener condiciones óptimas que protejan el adhesivo utilizado durante la aplicación de extensiones.

---

## Componentes

- ESP32
- Sensor DHT22
- Sensor MQ135
- Pantalla OLED I2C
- Módulo Relé 4 canales
- Ventilador 12V
- Humidificador ultrasónico
- LEDs indicadores
- Google Sheets

---

## Funciones

- Monitoreo en tiempo real
- Visualización en OLED
- Indicadores LED
- Activación automática de actuadores
- Almacenamiento de datos en Google Sheets
- Conectividad WiFi
